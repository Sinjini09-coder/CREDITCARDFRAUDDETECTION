# 💳 Credit Card Fraud Detection (Machine Learning)

## 📌 Overview

This project detects fraudulent credit card transactions using machine learning.
It analyzes transaction patterns and classifies them as *fraudulent or legitimate*.

---

## 📊 Dataset

* Credit Card Fraud Detection Dataset
* ~284,807 transactions
* Highly imbalanced dataset

📥 Download dataset from Kaggle and place it in the project folder before running.

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

---

## 🔍 Steps Performed

1. Data preprocessing
2. Exploratory Data Analysis (EDA)
3. Model training
4. Model evaluation

---

## 📈 Model Performance

* Accuracy: ~99%
* Used confusion matrix and classification report
